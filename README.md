# React + TypeScript + Vite

To run this project, install it's dependencies and then type the following command:

npm install
npm run dev

## About

This project is the homepage of a store that sells videogame related stuff (consoles, televisions, headsets, etc...)

## Features

- **(Mobile) Menu**: A side menu that opens when you click on it and shows the various links to different pages (**other pages not implemented**)
- **(Mobile) Footer Menu**: A menu to hide footer content when on mobile.
- **Newsletter Message**: A newslleter that sends a success message when you type a valid email.
- **AddToCart**: It was implemented a add to cart button inside the shelfs, when you click on it a item is added to the minicart (check on top of the page).
- **Minicart**: It shows every product added to the cart with every necessary information. It has a feature where you can add more quantity to those products and another feature where you can delete de item from the cart. Every product is saved on local storage.
